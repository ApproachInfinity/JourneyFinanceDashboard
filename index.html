CSV Format Requirements:

Please prepare your CSV file as follows:

  1. Rename your column headers to match exactly:
    • date - Transaction date (YYYY-MM-DD)
    • amount - Transaction amount (positive or negative number)
    • description - Transaction description
  2. Remove any other columns (account numbers, reference numbers, etc.)
  3. Ensure dates are in YYYY-MM-DD format
  4. Remove any currency symbols from amounts
Example CSV format:
date,amount,description
2024-01-15,-50.00,Grocery shopping
2024-01-16,1200.00,Salary deposit
